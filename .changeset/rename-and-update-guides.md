- 🐦 **更名为 BIRD.nvim 并补充升级指南** / **Rename to BIRD.nvim and add upgrade guides**

仓库已由 BIRD2.nvim 更名为 BIRD.nvim，并新增 lazy.nvim、packer.nvim、原生
packages 与现有检出的中英文升级步骤；共享子模块改用 BIRD.vim，`bird2`
filetype、Lua module、命令和运行时路径保持兼容。

The repository was renamed from BIRD2.nvim to BIRD.nvim, with bilingual upgrade
steps for lazy.nvim, packer.nvim, native packages, and existing checkouts. The
shared submodule now uses BIRD.vim, while the `bird2` filetype, Lua module,
commands, and runtime paths remain compatible.
